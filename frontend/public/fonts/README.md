# Font Installation Instructions

## Rabar_019 Font Setup

To use the Rabar_019 font in your website:

1. **Place your font files** in this directory (`public/fonts/`)

2. **Required font formats:**
   - `Rabar_019.ttf` (TrueType Font)
   - `Rabar_019.woff` (Web Open Font Format)
   - `Rabar_019.woff2` (Web Open Font Format 2 - recommended)

3. **The CSS is already configured** in `src/app/globals.css`

4. **If you don't have the font files**, the website will fall back to:
   - Segoe UI
   - Tahoma
   - Arial
   - Default sans-serif

## How to Add Font Files

Simply copy your Rabar_019 font files to this directory:
```
public/fonts/Rabar_019.ttf
public/fonts/Rabar_019.woff
public/fonts/Rabar_019.woff2
```

The website will automatically use them once added!
